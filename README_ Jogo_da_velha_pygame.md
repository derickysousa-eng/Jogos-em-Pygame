# Jogos-em-Pygame
# Jogo da Velha em Python

Um jogo da velha desenvolvido em **Python** utilizando a biblioteca **Pygame**.

## Funcionalidades

* Interface gráfica.
* Dois jogadores (X e O).
* Alternância automática de turnos.
* Detecção de vitória.
* Linha destacando a combinação vencedora.
* Detecção de empate.
* Botão **Reset** para reiniciar a partida.
* Atalho **R** para reiniciar o jogo.

## Tecnologias

* Python 3
* Pygame

## Como executar

1. Instale o Python.
2. Instale o Pygame:

```bash
pip install pygame
```

3. Execute o arquivo principal:

```bash
python nome_do_arquivo.py
```

## Controles

* **Clique com o mouse** para marcar uma posição no tabuleiro.
* **Botão Reset** para iniciar uma nova partida.
* **Tecla R** para reiniciar o jogo.
